## Opis szablonu

Kanwa Propozycji Wartości (ang. Value Proposition Canvas) w układzie klasycznym według Osterwaldera składa się z dwóch głównych elementów: koła po prawej stronie (Profil Klienta) oraz kwadratu po lewej stronie (Mapa Wartości). Koło jest podzielone na trzy strefy: Zadania Klienta (górna część), Bóle (dolna lewa), Korzyści (dolna prawa). Kwadrat jest podzielony na trzy strefy: Produkty i Usługi (środek), Uśmierzacze Bólu (dolna część), Kreatory Korzyści (górna część). Każda strefa posiada własny kolor karteczek, który jest z góry przypisany przez prowadzącego. Cały szablon musi być skonfigurowany przez prowadzącego przed zajęciami — grupy wypełniają go karteczkami, nie tworzą układu od zera. Na etapie startu ćwiczenia Mapa Wartości (kwadrat po lewej stronie) jest w całości zasłonięta etykietą "NIE ZACZYNAJCIE TUTAJ" — odsłonięcie następuje dopiero na Kroku 5 facilitacji, po zakończeniu pracy nad Profilem Klienta.

---

## Specyfikacja układu

### Ogólna struktura przestrzenna

Tablica Canva podzielona jest na strefy — jedną na grupę (Grupa A, Grupa B itd.). Każda strefa grupy zawiera pełny, kompletny egzemplarz Kanwy Propozycji Wartości.

Kanwa w obrębie każdej strefy grupowej:

```
+-----------------------------+-----------------------------+
|                             |                             |
|    MAPA WARTOŚCI            |    PROFIL KLIENTA           |
|    (kwadrat, lewa strona)   |    (koło, prawa strona)     |
|                             |                             |
|  +----------+----------+   |   +---------------------+   |
|  | Kreatory  | Produkty |   |   |   Zadania Klienta   |   |
|  | Korzyści  |    i     |   |   |     (góra koła)     |   |
|  |          | Usługi   |   |   +---------------------+   |
|  +----------+          |   |   |         |           |   |
|  | Uśmierzacze Bólu    |   |   |  Bóle   | Korzyści  |   |
|  |                     |   |   | (lewo)  | (prawo)   |   |
|  +---------------------+   |   +---------+-----------+   |
|                             |                             |
+-----------------------------+-----------------------------+
```

### Etykieta "NIE ZACZYNAJCIE TUTAJ"

Na początku ćwiczenia cała lewa strona (Mapa Wartości — kwadrat) jest przykryta dużym prostokątem lub naklejką w kontrastowym kolorze (np. jaskrawy żółty lub pomarańczowy) z napisem:

> **NIE ZACZYNAJCIE TUTAJ**
> Mapa Wartości zostaje odsłonięta dopiero w Kroku 5.
> Najpierw: wypełnijcie cały Profil Klienta (koło po prawej stronie).

Etykieta ta obejmuje **wszystkie trzy strefy** Mapy Wartości: Produkty i Usługi, Uśmierzacze Bólu oraz Kreatory Korzyści. Jest usuwana (lub przesuwana poza obszar roboczy) przez prowadzącego dopiero po ogłoszeniu Kroku 5, gdy grupy zakończyły pracę nad Profilem Klienta i dokonały rangowania Bólów i Korzyści.

**Czas zasłonięcia**: od początku ćwiczenia aż do Kroku 5 (po ok. 25 minutach od startu, tj. ok. 14:50).

## Tabela kolorów

| Strefa | Element | Kolor karteczki |
|--------|---------|-----------------|
| Profil Klienta — **Zadania Klienta** | Co klient chce osiągnąć? | Żółty |
| Profil Klienta — **Bóle** | Frustracje, ryzyka, przeszkody | Czerwony |
| Profil Klienta — **Korzyści** | Pożądane wyniki i korzyści | Zielony |
| Mapa Wartości — **Produkty i Usługi** | Co oferuje marka? | Szary |
| Mapa Wartości — **Uśmierzacze Bólu** | Jak marka usuwa Bóle? | Pomarańczowy |
| Mapa Wartości — **Kreatory Korzyści** | Jak marka tworzy Korzyści? | Niebieski |

> **Uwaga**: Kolory są z góry ustalone przez prowadzącego i nie podlegają zmianie przez uczestników. Stała kolorystyka umożliwia natychmiastowe wizualne odczytanie wypełnienia kanwy przez całą salę.

## Pre-fill: Przykład roboczy — InPost (Profil Klienta)

**Persona**: "Klient online w Warszawie, który nie odbiera paczek w domu"

Poniżej znajdują się karteczki pre-wypełnione przez prowadzącego przed zajęciami. Obejmują wyłącznie Profil Klienta (koło po prawej stronie) — Mapa Wartości pozostaje zasłonięta etykietą "NIE ZACZYNAJCIE TUTAJ".

### Zadania Klienta (2 karteczki — żółte)

| # | Treść karteczki (po polsku) |
|---|---------------------------|
| 1 | Odebrać paczkę bez konieczności czekania w domu na kuriera |
| 2 | Zamawiać online bez obawy o nieudaną dostawę i ponowne próby |

### Bóle (2 karteczki — czerwone)

| # | Treść karteczki (po polsku) |
|---|---------------------------|
| 1 | Kurier nie zastaje mnie w domu — przesyłka wraca do nadawcy lub czekam na kolejną wizytę |
| 2 | Wąskie okna czasowe dostaw — nie mogę zaplanować dnia pracy wokół przyjazdu kuriera |

### Korzyści (2 karteczki — zielone)

| # | Treść karteczki (po polsku) |
|---|---------------------------|
| 1 | Odbieram paczkę kiedy chcę — o każdej porze, 7 dni w tygodniu, bez uzgadniania terminu |
| 2 | Paczkomat jest 7 minut od mojego domu — odbiór zajmuje 2 minuty, nie tracę czasu |

> **Łącznie**: 6 karteczek pre-wypełnionych (2 na każdą strefę Profilu Klienta). Mapa Wartości (kwadrat) nie zawiera żadnych pre-wypełnionych karteczek — jest zasłonięta etykietą "NIE ZACZYNAJCIE TUTAJ".

## Instrukcje dla uczestników (po polsku)

Poniższe kroki są wyświetlane uczestnikom na slajdzie instruktażowym lub dostępne jako tekst na samej tablicy Canva (w dedykowanej strefie instrukcji).

---

**Ćwiczenie 3: Kanwa Propozycji Wartości**
**Czas: 50 minut | Narzędzie: Canva Whiteboard**

---

**Krok 1 — Orientacja i przykład (5 minut)**

Prowadzący pokazuje przykład roboczy InPost dla persony "Klient online w Warszawie, który nie odbiera paczek w domu". Zapoznajcie się z przykładem — zobaczcie, jak wygląda wypełniony Profil Klienta. Zwróćcie uwagę: Mapa Wartości jest jeszcze zasłonięta. Zaczynamy ZAWSZE od koła po prawej stronie.

---

**Krok 2 — Wybór segmentu klienta (3 minuty)**

Jako grupa zdecydujcie: dla którego konkretnego segmentu klientów waszej marki budujecie dziś kanwę?

Bądźcie konkretni. Nie "klienci e-commerce" — tylko "mama dwójki dzieci w Krakowie, która zamawia online raz w tygodniu i potrzebuje elastycznej dostawy." Im bardziej konkretna persona, tym lepsze wyniki.

Zapiszcie opis persony na karteczce i przyczepcie ją w górnej części waszej strefy grupowej.

---

**Krok 3 — Profil Klienta — praca solo (15 minut)**

Każda osoba pracuje samodzielnie i dodaje karteczki do Profilu Klienta (koło po prawej stronie). Nie dotykajcie Mapy Wartości — jest jeszcze zasłonięta.

- **Zadania Klienta** (żółte karteczki): Co ta persona chce osiągnąć? Jakie ma zadania funkcjonalne (np. "odebrać paczkę"), społeczne (np. "wyglądać profesjonalnie") i emocjonalne (np. "czuć się zorganizowaną")?
- **Bóle** (czerwone karteczki): Jakie frustracje, ryzyka i przeszkody napotyka ta persona? Co ją denerwuje? Czego się boi?
- **Korzyści** (zielone karteczki): Jakich wyników i korzyści oczekuje ta persona? Co by ją pozytywnie zaskoczyło?

Każda osoba dodaje własne karteczki — nawet jeśli ktoś z grupy napisał już coś podobnego. Liczy się różnorodność perspektyw.

---

**Krok 4 — Konsolidacja Profilu Klienta (5 minut)**

Jako grupa przejrzyjcie wspólnie Profil Klienta (koło):

1. Usuńcie oczywiste duplikaty (przesuńcie je na siebie).
2. Wybierzcie metodą głosowania (kropka lub gwiazdka w Canva) **3 najważniejsze Bóle** i **3 najważniejsze Korzyści** dla tej persony.
3. Wyróżnione Bóle i Korzyści będą kotwicą dla Mapy Wartości w kolejnym kroku.

---

**Krok 5 — Odsłonięcie Mapy Wartości (2 minuty)**

Prowadzący usuwa etykietę "NIE ZACZYNAJCIE TUTAJ" z lewej strony kanwy.

Spojrzyjcie na wasze 3 najważniejsze Bóle i 3 najważniejsze Korzyści. Teraz pytanie: co wasza marka robi, żeby odpowiedzieć na te konkretne potrzeby?

---

**Krok 6 — Mapa Wartości — praca solo (15 minut)**

Każda osoba pracuje samodzielnie i dodaje karteczki do Mapy Wartości (kwadrat po lewej stronie).

- **Produkty i Usługi** (szare karteczki): Co konkretnie oferuje wasza marka? Jakie produkty, usługi, funkcje?
- **Uśmierzacze Bólu** (pomarańczowe karteczki): Jak wasza marka adresuje konkretne Bóle zaznaczone w Kroku 4? (Każdy Uśmierzacz Bólu powinien być powiązany z konkretnym Bólem z koła.)
- **Kreatory Korzyści** (niebieskie karteczki): Jak wasza marka tworzy konkretne Korzyści zaznaczone w Kroku 4? (Każdy Kreator Korzyści powinien być powiązany z konkretną Korzyścią z koła.)

---

**Krok 7 — Analiza luk i prezentacja (5 minut)**

Porównajcie koło i kwadrat:
- Które Bóle z Profilu Klienta nie mają żadnego Uśmierzacza Bólu po lewej stronie?
- Które Korzyści z Profilu Klienta nie mają żadnego Kreatora Korzyści po lewej stronie?

Te luki to **strategiczne szanse** — obszary, w których wasza marka jeszcze nie odpowiada na potrzeby klienta.

Każda grupa dzieli się z salą **jedną najważniejszą luką** lub jedną najsilniejszą dopasowaną parą (Ból + Uśmierzacz Bólu lub Korzyść + Kreator Korzyści).
